Introdução

O flexbox do CSS, é um modelo de layout que facilita o design de interfaces web responsivas e flexíveis, permitindo o posicionamento e o alinhamento eficiente de elementos em relação ao espaço disponível, simplificando o desenvolvimento de layouts complexos.

display: flex

A primeira propriedade que precia ser declarada para que o Flexbox funcione!

Algumas propriedades como flex-direction, justify-content e align-content já vem com os valores padrâo.

justify-content: flex-start

align-content: flex-start

flex-direction

Determina a direção de alinhamento dos itens flexíveis dentro de um contêiner flexível.

Valores:

  • row: esquerda para a direita. (padrão)
  • row-reverse: direita para a esquerda
  • column: mesmo que row, mas de cima para baixo
  • column-reverse: mesmo que row-reverse mas de baixo para cima

flex-wrap

Essa propriedade permite que os itens quebrem para uma linha seguinte conforme necessário, alterando seu comportamento padrão.

Valores:

  • nowrap: os itens ficarão em uma só linha. (padrão)
  • wrap: os itens vão quebrar em múltiplas linhas, de cima para baixo
  • wrap-reverse: os itens vão quebrar em múltiplas linhas, de baixo para cima

justify-content

Define o alinhamento dos itens ao longo do eixo principal e auxilia na distribuição do espaço disponível dentro do contêiner.

Valores:

  • flex-start: os ítens são alinhados junto à borda de início. (padrão)
  • flex-end: os ítens são alinhados junto à borda final.
  • center: os ítens são centralizados na linha.
  • space-between: os ítens são distribuídos de forma igual ao longo da linha.
  • space-around: os ítens são distribuídos na linha com o mesmo espaçamento entre eles.
  • space-evenly: os ítens são distribuídos de forma que o espaçamento entre quaisquer dois itens da linha seja igual.

align-itens

Estabelece o comportamento padrão para o alinhamento dos itens flexíveis de acordo com o eixo transversal.

Valores:

  • stretch: estica os ítens para preencher o container. (padrão)
  • flex-start: ítens são posicionados no início do eixo transversal.
  • flex-end: ítens são posicionados no final do eixo transversal.
  • center: ítens são centralizados no eixo transversal.
  • baseline: ítens são alinhados de acordo com suas baselines.

align-content

Organiza as linhas dentro de um contêiner flexível quando há espaço extra no eixo transversal, semelhante ao modo como justify-content alinha os itens individuais ao longo do eixo principal.

Valores:

  • stretch: ítens em cada linha esticam para ocupar o espaço remanescente entre elas. (padrão)
  • flex-start: ítens alinhados com o início do container.
  • flex-end: ítens alinhados com o final do container.
  • center: ítens centralizados no container.
  • space-between: ítens distribuídos igualmente; a primeira linha junto ao início do container e a última linha junto ao final do container.
  • space-around: ítens distribuídos igualmente com o mesmo espaçamento entre cada linha.
  • space-evenly: ítens distribuídos igualmente com o mesmo espaçamento entre eles.

flex-flow

Esta propriedade é uma forma abreviada para a escrita das propriedades flex-direction e flex-wrap.

Valores:

  • row nowrap: Coloca o conteúdo em linha e não permite a quebra de linha.
  • row wrap: Coloca o conteúdo em linha e permite a quebra de linha.
  • column: mesmo que row, mas de cima para baixo
  • column-reverse: mesmo que row-reverse mas de baixo para cima
